What is one benefit of using a parallel circuit?

(A) If one bulb goes out, all the rest also go out.

(B) It is very cheap to make due to the low number of parts used.

(C) ** If one item goes down, it does not affect the others.

(D) It is impossible to break.

EXPLANATIONS BELOW

Concept note-1: -Total resistance in parallel circuits decreases, hence, a greater current may be drawn from the cell. If one electrical device is damaged, it will not affect the other devices connected parallel, hence the other devices continue to work properly.

Concept note-2: -Two bulbs in a simple parallel circuit each enjoy the full voltage of the battery. This is why the bulbs in the parallel circuit will be brighter than those in the series circuit. Another advantage to the parallel circuit is that if one loop is disconnected, then the other remains powered.

Concept note-3: -Parallel circuits provide more than one current path between any two points. These circuits have the advantage of making each component effectively invisible to the other components. This makes each load (electrical component, like a hairdryer) independent.

Concept note-4: -Answer. Advantages of parallel combination over series combination are:1) In parallel combination each appliance gets the full voltage. 2) If one appliance is switched on/of others are not affected. 3) The parallel circuit divide the current through the appliances.
