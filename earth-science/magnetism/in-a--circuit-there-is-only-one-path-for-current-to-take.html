In a ____ circuit, there is only one path for current to take.

(A) attract

(B) ** series

(C) fuse

(D) increase

EXPLANATIONS BELOW

Concept note-1: -A series circuit has only one conducting path for the electrons that move through, whereas a parallel circuit has multiple conducting paths.

Concept note-2: -Since there is only one path for electron flow in a series circuit, the current is the same magnitude at any point in the circuit. The total current in a series circuit is the same as the current through any resistance of the circuit.

Concept note-3: -The current in a series circuit goes through every component in the circuit. Therefore, all of the components in a series connection carry the same current. A series circuit has only one path through which its current can flow.

Concept note-4: -In summary, a series circuit is defined as having only one path through which current can flow. From this definition, three rules of series circuits follow: all components share the same current; resistances add to equal a larger, total resistance; and voltage drops add to equal a larger, total voltage.

Concept note-5: -A series circuit is a circuit that has only one pathway.
