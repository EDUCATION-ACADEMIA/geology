Which variable do you measure?

(A) independent variable

(B) ** dependent variable

EXPLANATIONS BELOW

Concept note-1: -Dependent variables are the measured behaviors of participants. They are dependent because they “depend on” what the participants do. The most obvious example is text entry speed, measured, for example, in words per minute.

Concept note-2: -In an experiment when you make changes in the Independent Variables your aim is to measure the changes it causes to the Dependent Variables. Remember that the dependent variable is affected by the changes you make in the independent variable.

Concept note-3: -The measurable variable, as the name suggests, is the variable that is measured in an experiment. It is the dependent variable (DV), which depends on changes to the independent variable (IV). Any experiment studies the effects on the DV resulting from changes to the IV.

Concept note-4: -The interval variable is a measurement variable that is used to define values measured along a scale, with each point placed at an equal distance from one another.
