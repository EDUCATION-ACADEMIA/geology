A ship is mapping the structure of the ocean floor using sonar technology. Which describes the basic method used in this type of mapping?

(A) A radio wave is sent from the ship and reflects off the ocean floor back to a receiver on the ship.

(B) ** A sound wave is sent from the ship and reflects off the ocean floor back to a receiver on the ship.

(C) A radio wave is sent by the ship and is absorbed by the ocean floor.

(D) A sound wave is sent by the ship and is absorbed by the ocean floor.

EXPLANATIONS BELOW

Concept note-1: -A ship is mapping the structure of the ocean floor using sonar technology. Which describes the basic method used in this type of mapping? A radio wave is sent from the ship and reflects off the ocean floor back to a receiver on the ship.

Concept note-2: -Which is a principle that allows scientists to use sonar to map the ocean floor? Sound waves travel more quickly through gases than they do through liquids or solids.

Concept note-3: -Using Sonar. The first modern breakthrough in sea floor mapping came with the use of underwater sound projectors, called “sonar”, which was first used in World War I.

Concept note-4: -Here’s how it works. Multibeam sonar signals are sent out from the ship. With about 1500 sonar soundings sent out per second, multibeam “paints” the seafloor in a fanlike pattern. This creates a detailed “sound map” that shows ocean depth, bottom type, and topographic features.

Concept note-5: -Mapping the seafloor is very challenging, because we cannot use the same techniques that we would use on land. To map the deep ocean, we use a tool called a multibeam echo-sounder, which is attached to a ship or a submarine vessel.
