What is the definition of speed?

(A) The time it takes an objects to accelerate

(B) ** The distance traveled in a given amount of time.

(C) How long it takes for an object to reach its destination

EXPLANATIONS BELOW

Concept note-1: -Speed is defined as. The rate of change of position of an object in any direction. Speed is measured as the ratio of distance to the time in which the distance was covered. Speed is a scalar quantity as it has only direction and no magnitude.

Concept note-3: -The quantity which tells the distance an object travels in a certain time is called speed.

Concept note-5: -To solve for speed or rate use the formula for speed, s = d/t which means speed equals distance divided by time. To solve for time use the formula for time, t = d/s which means time equals distance divided by speed.
